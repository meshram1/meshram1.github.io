Graph Neural Network Surrogate for Airfoil Flow Field Prediction

Real-time CFD inference via mesh-graph message passing

using PyTorch Geometric · code →

Trained an encode-process-decode GNN in PyTorch Geometric to predict full 2D flow fields (u, v, p) directly from mesh-graph inputs. The trained network acts as a real-time surrogate for computationally expensive CFD solvers — the kind of inference speedup that turns design exploration from overnight sweeps into interactive iteration.

GPU-Accelerated Poisson Solver

Jacobi iterations on CUDA, 13× end-to-end speedup

built with CUDA · cuBLAS · code →

hpc cuda gpu cpp cfd

Implemented a GPU-accelerated Poisson solver using Jacobi iterations with a custom tensor abstraction for host/device memory management. Integrated cuBLAS for vector operations and optimized GPU memory access using ping-pong buffers. The result: 6× per-iteration speedup and 13× end-to-end over a CPU baseline.
